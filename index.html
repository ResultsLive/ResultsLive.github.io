This is a trial live results page for the Oxfordshire Track & Field League. There is no need to refresh the page as it will automatically check for new results every minute.

173 QuadKids & 297 U13-U20 athletes have declared for the match.

The match timetable can be found here

Click through the top menu links (the top menu will be displayed as 'Menu' on a mobile phone) to select the set of results you want ie U13-17 Scoring, U13-U20 Non Scoring or QuadKids. Each set of results can be filtered by age, sex, track events, field events or club (on a mobile phone click 'Options' to see these filters).

Clicking on the Power of 10 icon next to an athlete's name will take you through to that athlete's Power of 10 listing.

The 'U13-U20 Declarations' link lists all the athletes that have declared for a particular event.

Any queries on the results please talk to your team manager and they can then query with the scorers.
